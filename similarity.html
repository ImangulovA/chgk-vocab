Похожесть авторов: 4 взгляда

«Похожи» бывает разное, поэтому соседей можно искать четырьмя способами. Выберите автора и сравните, кого выдаёт каждый метод (как именно работает каждый, с живым демо про «буквы», — на странице Методы):

  • По лексике — общие характерные слова (log-odds отпечаток). «Пишут про похожее похожими словами».
  • По манере (Delta) — стилометрия по служебным словам. «Похоже устроена речь», без влияния темы.
  • По буквосочетаниям — символьные 3-граммы. Почерк на уровне морфологии.
  • По смыслу (нейросеть) — локальный doc2vec. Общее семантическое поле.

Три примера

Видно, что методы дают разных соседей: это и есть разные грани «похожести».

Данные и профили авторов: база вопросов ЧГК gotquestions.online. Метод «фирменных слов» — weighted log-odds (см. Как считаем). Дизайн — тема Pac-Man.
