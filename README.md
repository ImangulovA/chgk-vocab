# CHGK Lexicon — словарный запас авторов вопросов ЧГК

Корпусное исследование лексического разнообразия **авторов вопросов «Что? Где? Когда?»**,
по мотивам исследования рэперов Ильи Иноземцева ([rap.inozemtsev.lol](https://rap.inozemtsev.lol/)).

**Живая инфографика:** https://imangulova.github.io/chgk-vocab/ · дизайн — тема Pac-Man
([bergside/awesome-design-skills](https://github.com/bergside/awesome-design-skills/tree/main/skills/pacman)).

## Что считаем

Для каждого автора берём тексты и комментарии всех его вопросов, оставляем только
**кириллицу**, лемматизируем ([pymorphy3](https://github.com/no-plagiarism/pymorphy3))
и считаем метрику:

> **число уникальных лемм в первых 25 000 словах** корпуса автора

Окно в 25 000 слов — как у Иноземцева. Автор с корпусом < 25 000 слов показывается
«бледным» (метрика измерена на меньшем окне, надёжность ниже).

## Данные

- Источник: дамп базы вопросов ЧГК (gotquestions / db.chgk.info), один JSON на пакет.
- Масштаб: **6701 пакет, 436 454 вопроса, 9 430 авторов**.
- Дедуп по глобальному `id` вопроса; со-авторские вопросы засчитываются каждому соавтору.
- Порог отображения — 10 000 слов корпуса (359 авторов, из них **149 надёжных**).

Сам дамп в репозиторий не входит (чужой контент). Публикуются только агрегат
(`report.json`) и инфографика (`index.html`).

## Как пересобрать

Нужны `razdel` и `pymorphy3`.

```bash
python3 -m venv .venv && ./.venv/bin/pip install razdel pymorphy3
# путь к дампу задаётся в scripts/build_report.py (PACKS_DIR)
./.venv/bin/python scripts/build_report.py   # -> report.json
./.venv/bin/python scripts/build_html.py     # -> index.html (данные вшиты)
```

## Оговорки

Метрика измеряет **разнообразие словаря**, а не «ум» или качество вопросов: у прозаичных
формулировок и у авторов с большим числом коротких вопросов метрика естественно выше.
