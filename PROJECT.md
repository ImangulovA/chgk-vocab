# Лексикон ЧГК — словарный запас авторов вопросов

Корпусное исследование лексического разнообразия **авторов вопросов ЧГК**, по
аналогии с исследованием рэперов Ильи Иноземцева (rap.inozemtsev.lol) и нашим
[stand-up research](../standup_research/PROJECT.md).

Живой сайт: **https://imangulova.github.io/chgk-vocab/** · GitHub: **ImangulovA/chgk-vocab**

## Цель
По базе вопросов «Что? Где? Когда?» посчитать для каждого автора метрику
«у кого богаче словарь» и выложить интерактивную инфографику (self-contained
HTML, данные вшиты). Разрослось в мультистраничный сайт: рейтинг, фирменные
слова, родственные авторы, граф соавторства, слово/имя/место года, штампы.

## Данные
- Источник: дамп базы вопросов ЧГК (gotquestions / db.chgk.info), формат — один
  JSON на пакет/турнир: `~/Downloads/Quiz Packs/packs/*.json` (6701 пакет).
- Структура: pack -> `tours[]` -> `questions[]`. У вопроса есть `text`, `comment`,
  `answer`, `source` и список `authors` (`id`, `name`, `gender`) и `editors`.
- Масштаб: **436 454 вопроса, 9 430 авторов**.

## Методология (1:1 с Иноземцевым/стендапом, + наши правки)
- Корпус автора = `text` + `comment` всех его вопросов, в хронологическом порядке
  (по `endDate`/`pubDate` вопроса, затем по id).
- Только **кириллические** токены (латиница/цифры/имена собственные латиницей
  выбрасываются). Токенизация — просто по кириллице; лемматизация — `pymorphy3`.
- Метрика: число **уникальных лемм в первых 25 000 словах** корпуса (как у
  Иноземцева). Корпус < 25 000 слов = «бледный круг» (метрика на меньшем окне).
- Дедуп по глобальному `id` вопроса (один вопрос бывает в нескольких пакетах).
- Со-авторские вопросы засчитываются **каждому** из перечисленных авторов.
- Порог отображения: 10 000 слов корпуса (359 авторов).

### Фиксы качества данных (важно; применены в build_report.py И compute_features.py)
- **Фильтр языка.** Белорусские/украинские вопросы выкидываем по буквам `і ї є ґ ў`
  (которых в русском алфавите нет; порог >3% от кириллицы). НЕ по словам — гомограф
  «гэта»(обувь) в русском вопросе остаётся. НЕ по стране автора: Беларусь/Украина
  пишут массу вопросов по-русски (Климович №1 — белорус). Метадата gotquestions
  для страны недоступна: `/person/<id>` отдаёт только `city={id,name}`, API под 401.
  Отсеяно ~438 вопросов (0.1%).
- **Снятие ударений.** В корпусе ~13k знаков `U+0301` (COMBINING ACUTE): «Дека́рт»
  без чистки рвётся на «дека»+«рт». Снимаем `U+0301/U+0300` ДО токенизации, точечно
  (не через NFD — иначе ломаются `ё`/`й`).
- **Лемма раз на вопрос.** Частота = в скольких вопросах встречается (uniq=set на
  вопрос). Иммунно к спаму типа «гек ×100» в одном вопросе.

## Пайплайн (`scripts/`)
- `build_report.py` — два прохода по всем пакетам (pass1 = подсчёт слов и отбор
  отображаемых, pass2 = сбор токенов) + лемматизация первых 25k -> `report.json`
  (метрика словаря + `pid` профиля gotquestions).
- `compute_features.py` — тяжёлые «фишки»: weighted log-odds (фирменные слова:
  темы + стиль), родственные авторы, граф соавторства, слово/имя/гео года,
  штампы/клише. Кэширует агрегаты в `scripts/_raw.pkl` (пересчёт: `--fresh`; иначе
  быстрый re-emit из кэша за секунды). Пишет: дополняет `report.json` +
  `coauthors.json`, `years.json`, `cliches.json`, `stoplist.json`.
  - **Фирменные слова:** weighted log-odds с приором Дирихле (Monroe et al. 2008);
    деление на «темы» (Surn/Name/Geox/Orgn по тегам pymorphy) и «стиль» (нарицательное).
  - **Родственные авторы (3):** косинус **стилевых отпечатков** = вектор положительных
    z-оценок log-odds по нарицательным словам. НЕ tf-idf: иначе плодовитые авторы
    (Мерзляков/Коробейников/Кудрявцев) выпадали в соседи почти всем (хаб-эффект).
  - **ЧГК-жаргон** (икс/замена/раздатка/бескрылка/жанры/носители + склонения ИКС
    кс/кса/ксом) — в META-стоп-листе (155 слов); выгружается в `stoplist.json` и
    показывается раскрывашкой на years.html.
- `build_site.py` — собирает **6** self-contained страниц из JSON (общая тема/навигация,
  светлая/тёмная): `index.html` (рейтинг + раскрытие: слова-маркеры/темы/родственники),
  `years.html` (слово/имя/место года + список исключённых слов), `coauthors.html`
  (canvas force-граф + рейтинг «у кого больше всего связей»), `cliches.html` (штампы),
  `method.html` (объяснение log-odds), `pipeline.html` («Кухня» — данные+скрипты).

Порядок: `build_report.py` -> `compute_features.py` -> `build_site.py`.
Venv из стендап-проекта (razdel+pymorphy3):
```
V=/Users/imangulov/Desktop/standup_research/.venv/bin/python3
$V scripts/build_report.py            # report.json (метрика словаря + pid)
$V scripts/compute_features.py --fresh # фичи (--fresh = пересчёт кэша; без него из _raw.pkl)
$V scripts/build_site.py              # все 6 страниц
```
(`build_html.py` — устаревший одностраничный сборщик, оставлен для истории.)

## Результаты (2026-07-09, после фиксов качества)
- Корпус после фильтра: **17,3 млн** учтённых лемм-вхождений (doc-freq), ~438 бел/укр
  вопросов отсеяно.
- Отображаются 359 авторов, из них **148 надёжных** (≥25 000 слов).
- Топ по богатству словаря: Леонид Климович (6809), Александр Мудрый, Евгений Кононенко,
  Сергей Донецков, Борис Бурда.
- Слово года: римлянин (90-е) → коронавирус (2020) → пандемия (2021) → ведьмак (2022)
  → нейросеть (2025); место года — спортивные города (Сочи/Лестер/Ливерпуль).
- Больше всего связей: Анатолий Белкин (108 соавторов), Богословский, Выменец, Кноп.
